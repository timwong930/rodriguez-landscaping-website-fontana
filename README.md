# Rodriguez Landscaping website starter

## Public business snapshot
- **Company:** Rodriguez Landscaping
- **Location:** Fontana, CA
- **Category:** Landscaping
- **Website:** https://www.rodriguezlandscapingservice.com/
- **Social:** none found in the public scan

## What they do
A landscaping and hardscape business serving the Inland Empire with an emphasis on budget-friendly service and drought-tolerant work.

## Why this looks like a good opportunity
- The homepage has repetitive copy and awkward grammar, which makes the brand feel less polished than the services deserve.
- There is a useful service mix already, but the site would benefit from separate pages for hardscape, drought-tolerant landscaping, and maintenance.
- A cleaner before/after gallery and simpler CTAs could improve both trust and conversion.

## Suggested outreach / redesign angle
Pitch a copy-and-structure cleanup with service-specific pages and a more professional visual system for Fontana and nearby cities.

## Starter site included in this repo
This repo includes a simple, responsive starter site in `index.html` and `styles.css` built only from public information.

### Draft structure
- **Landscaping** — General residential yard work and cleanups.
- **Hardscape** — Patios, stone, and other durable outdoor features.
- **Drought-tolerant yards** — Water-smart plantings and lower-maintenance layouts.
- **Reviews and booking** — A simple flow that helps homeowners request estimates.
